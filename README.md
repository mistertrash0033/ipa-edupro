# EduPro — iOS Project

## Requirements
- Xcode 15+, iOS 15+
- CocoaPods (`sudo gem install cocoapods`)

## Setup
1. Run `pod install` in this directory
2. Open `.xcworkspace` (NOT `.xcodeproj`)
3. Set your development team in Signing settings
4. Add your app icons to `Assets.xcassets/AppIcon.appiconset/`

## Firebase
Place `GoogleService-Info.plist` in `Sources/` (already included if you uploaded it)

## Config
- Bundle ID: `com.softprosolutions.edupro`  Min iOS: 15  URL: https://softprosolutions.com/edupro_mobile/index  Version: 1.0.0
